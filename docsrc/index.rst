PyBlast Overview
================


.. toctree::
   :maxdepth: 2
   :hidden:

   developer/api_reference

Pyblast is a wrapper around NCBI's BLAST tools. Handles circular sequences.

**Installation**

Install pyblastbio with pip:

.. code-block:: bash

    pip install pyblastbio

Then install NCBI's BLAST:

.. code-block:: bash

    pyblast install

For help with installation

.. code-block:: bash

    pyblast install -- --help
