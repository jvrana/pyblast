classmethod add_records(records, seq_db, span_origin=True)[source]¶

Adds records to the local sequence database (SeqRecordDb). If self.span_origin=True, then in addition to adding the original SequenceRecord to the database, sequences will be pseudocircularized (concatenated with itself) if they are detected to be circular (via the ‘topology’ annotation’) so that Blast can properly align sequences over the origin. Only sequences that were linear or the pseudociruclarized sequences are used in the alignment. After running blast and obtaining results, span indices are automatically corrected to account for the original pseudocircularization.

Parameters

  • records (list) – list of SeqRecords annotated with ‘topology’ being ‘linear’ or ‘circular’

  • seq_db (SeqRecordDB) – optional SeqRecordDB to use

  • span_origin (bool) – whether to search the sequence as a circular plasmid

Returns

list of keys to use in the alignment procedure
