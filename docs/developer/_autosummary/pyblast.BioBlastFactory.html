pyblast.BioBlastFactory¶

class pyblast.BioBlastFactory(seq_db=None, span_origin=True, config=None)[source]¶

Bases: object

Instance factory for BioBlast instance that share the same SeqRecordDB.

Usage:

factory = BioBlastFactory()
factory["plasmid_templates"] = records1
factory["primers"] = records2
factory["queries"] = records3

blast1 = factory("plasmid_templates", "queries")
blast2 = factory("primers", "queries")

Initialize a new BioBlastFactory.

Parameters

seq_db (SeqRecordDB) – the optional SeqRecordDB. If not provided, a new one will be created.
